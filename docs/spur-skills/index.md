# spur-skills

`spur-skills` is the command-line helper for installing and maintaining the
shared skills used across the SPUR and SCPC package ecosystem. It gives your
agent all necessary information on the ecosystem to get you set up.

- [Get started](get-started.md): install the tool and run the core commands
- [How it works](how-it-works.md): understand what the tool writes and why
